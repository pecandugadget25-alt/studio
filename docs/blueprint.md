# **App Name**: ETHNO-ARITH

## Core Features:

- Dashboard Numerasi & Gamifikasi: Panel utama siswa untuk memantau Skor Numerasi, XP, Level, serta Koleksi Badge seperti 'Ahli Geometri Batik' dan 'Penjaga Warisan'.
- Sistem Pembelajaran Adaptif: Menggunakan tool AI untuk menyesuaikan tingkat kesulitan kuis (Mudah, Sedang, Sulit) secara real-time berdasarkan akurasi jawaban siswa.
- Modul Eksplorasi Etnomatematika: Konten interaktif yang menghubungkan Batik, Candi, dan Masjid dengan konsep Geometri serta Operasi Hitung.
- Viewer Scan Augmented Reality: Placeholder kamera dan viewer AR untuk visualisasi 3D bangun ruang candi dan motif batik di lingkungan nyata.
- Sistem Monitoring & Analisis Guru: Dashboard bagi guru untuk melihat rekap nilai, progress belajar siswa, dan statistik performa per modul secara mendetail.
- Integrasi Database Firestore: Sinkronisasi data real-time untuk riwayat belajar, progres kuis, tantangan harian, dan manajemen profil pengguna menggunakan Firestore.
- Manajemen Konten Admin: Antarmuka administrator untuk mengelola bank soal kuis, modul pembelajaran, dan verifikasi pengguna.

## Style Guidelines:

- Warna Primer: Terracotta hangat (#B84D35) yang terinspirasi dari bata candi dan pewarna alami batik nusantara.
- Warna Latar: Krem gading pucat (#FAF7F5) untuk kenyamanan membaca dalam waktu lama.
- Warna Aksen: Emas Budaya (#D9A441) untuk elemen gamifikasi seperti lencana (badge) dan progres penting.
- Pairing font 'Poppins' (sans-serif) untuk judul agar terasa modern dan ramah anak, dipadukan dengan 'PT Sans' untuk teks isi yang jelas dan profesional.
- Gunakan set ikon kustom yang menggabungkan elemen geometri matematika dengan ornamen dekoratif tradisional Indonesia.
- Layout responsif berbasis kartu dengan ruang putih yang cukup untuk memberikan fokus pada konten edukasi di mobile maupun desktop.
- Animasi transisi halus antar halaman modul dan efek 'micro-interaction' yang ceria saat siswa mendapatkan reward atau poin.
